About the job
Backend Django REST API Developer



Key Responsibilities:

Develop and maintain RESTful APIs using Django REST Framework
Design, optimize, and manage relational database schemas
Write clean, maintainable, and well-tested Python code
Participate in code reviews, testing, and CI/CD pipelines.
Collaborate closely with frontend developers and product stakeholders
Troubleshoot, debug, and resolve backend issues in production environments
Ensure high performance and responsiveness of backend systems.


Requirements:
Bachelor's degree in Computer Science or related field
3+ years of hands-on experience with Django/Django REST Framework
Strong Python programming skills
Experience in REST API design and implementation
Solid understanding of SQL and database design
Proficient with Git version control
Familiarity with testing frameworks (e.g., pytest, unittest)
Experience with Docker in development and deployment workflows
Redis and Celery experience for asynchronous tasks
Exposure to CI/CD pipelines and modern development workflows
